# SSD1306 0.96" OLED I2C

128x64 pixels, I2C interface, SSD1306 driver. 3.3V-5V compatible.

## Pinout (display facing you, pins at top)

```
  +---------------------+
  | GND  VCC  SCL  SDA  |
  |  o    o    o    o   |
  |                     |
  |                     |
  |    0.96 inch OLED   |
  |                     |
  |                     |
  | (O)             (O) |
  +---------------------+
```

## Pin Description

- GND -- Ground
- VCC -- Power, 3.3V or 5V
- SCL -- I2C clock
- SDA -- I2C data

## I2C Address

Default: 0x3C. Some modules use 0x3D. Check/change via solder jumper on back if present.

## Wiring to XIAO ESP32-S3

```
  SSD1306          XIAO ESP32-S3
  -------          -------------
  GND    --------> GND
  VCC    --------> 3V3
  SCL    --------> D5 (GPIO6, SCL)
  SDA    --------> D4 (GPIO5, SDA)
```

## Wiring to XIAO nRF52840

```
  SSD1306          XIAO nRF52840
  -------          -------------
  GND    --------> GND
  VCC    --------> 3V3
  SCL    --------> D5 (P0.05, SCL)
  SDA    --------> D4 (P0.04, SDA)
```
